Demonstrator Timetabling Tool documentation
===========================================

Welcome to the official documentation for the Demonstrator Timetabling Tool. This tool is designed to assist in managing and optimizing timetables, specifically tailored for educational environments. Whether you are a developer looking to understand the underlying code, or an end-user seeking guidance on how to utilize the tool effectively, this documentation provides all the necessary information.

In the following sections, you'll find detailed instructions on setting up your environment, populating test databases, exploring available API endpoints, and much more. Use the navigation menu to explore different topics, and feel free to search the documentation for specific information.
