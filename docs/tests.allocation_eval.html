The StrategyTests class contains test cases for various allocation strategies in the Django application.

Parameters:

  • setUpTestData (Class Method) -- Sets up the test data at the class level, including creating a superuser, populating default timeslots, skills, modules, sessions, and timetables.

  • generate_demonstrators (Static Method) -- Generates a specified number of demonstrators based on a multiplier.

  • setUp (Method) -- Initializes the test environment for each test case, including logging in as the admin user and resetting allocations.

  • time_for_allocation (Static Method) -- Measures and returns the time taken for batch allocation.
